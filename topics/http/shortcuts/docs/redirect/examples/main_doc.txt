Examples

You can use the redirect() function in a number of ways.
1. By passing some object; that objectâ€™s get_absolute_url() method will be called 
to figure out the redirect URL:

```
from django.shortcuts import redirect

def my_view(request):
    ...
    obj = MyModel.objects.get(...)
    return redirect(obj)
```

2. By passing the name of a view and optionally some positional or keyword arguments; 
the URL will be reverse resolved using the reverse() method:

```
def my_view(request):
    ...
    return redirect("some-view-name", foo="bar")
```

3. By passing a hardcoded URL to redirect to:

```
def my_view(request):
    ...
    return redirect("/some/url/")
```

This also works with full URLs:

```
def my_view(request):
    ...
    return redirect("https://example.com/")
```
